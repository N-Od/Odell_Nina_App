# Odell_Nina_App
Semester Project
# Start Small - Semester Project 2

## What Works
The Expo project runs successfully with 'npx expo start'.
The app includes three screens:
  - Home
  - Add
  - Momentum
Tab navigation works across all screens.

Every screen has actual draft material, not just placeholders.
- Basic layout and styling have been implemented.
- Screenshots of the app running are supplied.

## Current Features:

Users can add new chores.
Users can mark chores completed or incomplete.
Users can delete the chores.
The Home screen displays live progress.
The Momentum screen describes the app's purpose and how to utilize it.
